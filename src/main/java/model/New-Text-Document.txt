String
SET CHAVE VALOR
GET CHAVE VALOR

Hash
HSET CHAVE  SUBCHAVE VALOR
      SUBCHAVE VALOR
      SUBCHAVE VALOR

HGET CHAVE  SUBCHAVE

Lista - Sem ordenação com repetição
LPUSH  CHAVE VALOR1 VALOR2 ... VALORn
RPUSH  CHAVE VALOR1 VALOR2 ... VALORn
LRANGE CHAVE POSICAOINICIO TAMANHO

Set - Sem ordenação, sem repetição
SADD  CHAVE  VALOR1 VALOR2 ... VALORn
SMEMBER CHAVE POSICAOINICIO TAMANHO
SMEMBERS CHAVE

Sorted Sets - Com ordenação, se repetição valores, n valores por posição
ZADD  CHAVE POSICAO VALOR
ZRANGE CHAVE POSICAOINICIO TAMANHO

Exemplo

{"USUARIO":
  { "APELIDO" : "ZE" , "NOME" : "ZE DA SILVA" , "DATA CADASTRO" : "01/01/2000" },
  { "APELIDO" : "MARIA" , "NOME" : "MARIA DA SILVA" , "DATA CADASTRO" : "02/01/2000" },
  { "APELIDO" : "JAO" , "NOME" : "JAO DA SILVA" , "DATA CADASTRO" : "03/01/2000" }
 }

SADD APELIDOS ZE
SADD APELIDOS MARIA
SADD APELIDOS JAO

HSET USUARIO:ZE APELIDO ZE
HSET USUARIO:ZE NOME "ZE DA SILVA"
HSET USUARIO:ZE "DATA CADASTRO" "01/01/2000"


{"MENSAGEM":
   "IDMSG": 1, "TEXTO" : "BUENAS ?", "DATA ENVIO" : "01/01/2000 00:00", "REMETENTE" : "ZE" ,
   "DESTINATARIO" :
   [ { "SEQUENCIA" : 1 , "APELIDO" : "JAO" },
     { "SEQUENCIA" : 2 , "APELIDO" : "MARIA" }
   ]
   "RESPOSTAS" :
   [ { "SEQUENCIA" : 1 , "IDMSG" : 2 },
     { "SEQUENCIA" : 2 , "IDMSG" : 3 }
   ]
 }

HSET MENSAGEM:IDMSG:1 IDMSG 1
HSET MENSAGEM:IDMSG:1 TEXTO "BUENAS ?"
HSET MENSAGEM:IDMSG:1 "DATA ENVIO" "01/01/2000 00:00"
HSET MENSAGEM:IDMSG:1 REMETENTE ZE

RPUSH MENSAGEM:IDMSG:1:DESTINATARIOS JAO
RPUSH MENSAGEM:IDMSG:1:DESTINATARIOS MARIA

ZADD MENSAGEM:IDMSG:1:RESPOSTAS 0 2
ZADD MENSAGEM:IDMSG:1:RESPOSTAS 1 3
